What is Chronicle?

Chronicle is a major protocol upgrade to the BSV Blockchain activating at block height 943,816. It introduces the Original Transaction Digest Algorithm (OTDA) via the new CHRONICLE [0x20] sighash flag, expands the script number limit from 750 KB to 32 MB, removes malleability restrictions for v2+ transactions, and restores nine previously disabled opcodes including OP_VER, OP_SUBSTR, OP_2MUL, and OP_2DIV.
